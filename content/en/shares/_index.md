File sharing is one of the primary benefits of a NAS. TrueNAS helps foster collaboration between users through network shares.

TrueNAS allows users to create and configure Windows SMB shares, Unix (NFS) shares, Multiprotocol shares (SMB and NFSv4), block (iSCSI) shares, and NVMe-OF subsystems.
